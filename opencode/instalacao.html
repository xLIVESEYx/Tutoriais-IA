📦 Instalação do OpenCode

Escolha o método de instalação que melhor se adapta ao seu ambiente. O mais rápido é via script curl.

🚀 Script curl (Recomendado)

A forma mais simples e rápida. Funciona em macOS, Linux e Windows (via WSL).

curl -fsSL https://opencode.ai/install | bash

✅ OpenCode instalado globalmente
✅ Comando opencode disponível no terminal

💡 Diretório de instalação: O script respeita a ordem: $OPENCODE_INSTALL_DIR → $XDG_BIN_DIR → $HOME/bin → $HOME/.opencode/bin

📦 Gerenciadores de Pacote

npm

npm install -g opencode-ai@latest

Bun

bun install -g opencode-ai

pnpm

pnpm install -g opencode-ai

Yarn

yarn global add opencode-ai
⚠️ Importante: Remova versões anteriores a 0.1.x antes de instalar.

🍺 Homebrew (macOS e Linux)

Tap oficial (recomendado)

brew install anomalyco/tap/opencode

Sempre atualizado com os últimos releases.

Fórmula oficial do Homebrew

brew install opencode

Mantida pela equipe do Homebrew — atualizada com menos frequência.

Desktop App

brew install --cask opencode-desktop

🪟 Windows

💡 Recomendado: Use WSL (Windows Subsystem for Linux) para melhor performance e compatibilidade total.

Chocolatey

choco install opencode

Scoop

scoop install opencode

npm direto

npm install -g opencode-ai

Mise

mise use -g github:anomalyco/opencode

Docker

docker run -it --rm ghcr.io/anomalyco/opencode

Scoop Desktop

scoop bucket add extras
scoop install extras/opencode-desktop

🐧 Linux (Arch Linux)

Estável (pacman)

sudo pacman -S opencode

Última versão (AUR)

paru -S opencode-bin

❄️ Nix

nix run nixpkgs#opencode
# Ou para o branch dev mais recente:
# github:anomalyco/opencode
