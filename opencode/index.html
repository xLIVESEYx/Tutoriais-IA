O que é o OpenCode?

O OpenCode é um agente de IA open source (MIT) que ajuda você a escrever código no terminal, IDE ou desktop. Com mais de 187.000 estrelas no GitHub, 900 contribuidores e 7,5 milhões de desenvolvedores mensais, é o agente de código aberto mais popular do mercado.

Construído pela equipe do Anomaly Innovations (mesma equipe por trás do SST), o OpenCode suporta 75+ provedores de LLM, integração nativa com LSP para 40+ linguagens, multi-sessão em paralelo, compartilhamento de sessões e muito mais.

💡 Por que usar o OpenCode?
Diferente de ferramentas proprietárias como Claude Code ou Cursor, o OpenCode é 100% open source, não prende você a um único provedor de IA, roda localmente sem armazenar seu código e funciona em qualquer editor.

🎯 Principais Benefícios

🔌 Provider-Agnóstico

Use qualquer modelo de 75+ provedores: Claude, GPT, Gemini, Groq, DeepSeek, Ollama local e mais. Sem vendor lock-in.

🏗️ Arquitetura Cliente/Servidor

Servidor em Bun + Hono (TypeScript), cliente em Go (Bubble Tea). Sessões persistem independente do cliente. Conecte de qualquer dispositivo.

🖥️ Multi-Plataforma

TUI nativa no terminal, Desktop App (Tauri), Extensão VS Code/Cursor e Web UI. Escolha sua interface favorita.

🧠 LSP Nativo

40+ linguagens com diagnóstico em tempo real. O agente recebe erros do LSP e se corrige automaticamente sem intervenção humana.

🤖 Sistema de Agentes

Build (acesso total), Plan (read-only), Explore, Scout, General e agentes customizáveis com prompts e permissões próprias.

🛡️ Privacidade Primeiro

Nenhum código ou contexto é armazenado externamente. Tudo roda localmente. Suporte a fully offline com modelos locais.

🚀 Quickstart de 3 Passos

1. Instale o OpenCode

curl -fsSL https://opencode.ai/install | bash

Ou via npm: npm install -g opencode-ai

2. Conecte um Provedor

Execute opencode e depois /connect no TUI. Selecione OpenAI, Anthropic, GitHub Copilot ou 75+ outros provedores.

3. Comece a Codar

cd /path/to/seu/projeto
opencode

Execute /init para o OpenCode analisar seu projeto e criar um AGENTS.md personalizado.
