🏢 Enterprise

Use o OpenCode de forma segura e escalável em ambientes organizacionais. Privacidade, SSO e governança.

🔒 Privacidade Primeiro

🛡️ Garantia: O OpenCode não armazena nenhum código ou contexto. Toda indexação, diagnósticos e contexto de inferência LLM permanecem locais ao seu ambiente.

Proteção da Feature Share

{
  "share": "disabled"
}

Desabilite compartilhamento em sensitive projects. Pode ser configurado por projeto e versionado no Git.

⚙️ Deploy Empresarial

Config Centralizada

Config compartilhada para toda a organização com todos os usuários usando o mesmo setup.

SSO Integration

Integra com Okta, Azure AD, Google Workspace para autenticação.

AI Gateway Interno

Roteie todo tráfego por gateway interno. Desabilite provedores externos.

Self-Hosting

Compartilhe na sua própria infraestrutura se necessário.

🏛️ Políticas e Governança

Desabilitar Provedores

{
  "disabled_providers": ["openai", "gemini"]
}

Políticas de Uso (Experimental)

{
  "experimental": {
    "policies": [
      {
        "effect": "deny",
        "action": "provider.use",
        "resource": "openai"
      }
    ]
  }
}

Permissões Granulares por Projeto

{
  "permission": {
    "bash": {
      "*": "ask",
      "git push *": "deny",
      "rm *": "deny"
    },
    "edit": "ask"
  }
}

📋 Managed Settings (MDM)

Force configurações que os usuários não podem sobrescrever:

Plataforma Caminho
macOS/Library/Application Support/opencode/
Linux/etc/opencode/
Windows%ProgramData%\opencode
macOS MDM.mobileconfig via Jamf/FleetDM
