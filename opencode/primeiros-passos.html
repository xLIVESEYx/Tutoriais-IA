🚀 Primeiros Passos

Configure o OpenCode em 3 passos simples e comece a codar com IA.

1. Conecte um Provedor de IA

Execute opencode no terminal e use o comando /connect para adicionar um provedor.

opencode
/connect

Você verá uma lista de provedores disponíveis. Selecione um e siga as instruções.

💡 Recomendado para iniciantes:
OpenCode Zen — Modelos curados e testados pela equipe. Pague por uso.
OpenCode Go — Plano de baixo custo com modelos abertos populares.
GitHub Copilot — Use sua assinatura Copilot existente.
ChatGPT Plus/Pro — Use sua assinatura OpenAI.

Autenticação via OAuth

Para provedores como Anthropic (Claude Pro/Max), OpenAI (ChatGPT Plus/Pro) e GitHub Copilot, você pode autenticar via OAuth — o navegador abre automaticamente para autorização.

/connect
# Selecione "Login with X"
# Navegador abre → Autorize → Pronto!

Usando API Key

Para provedores como DeepSeek, Groq, OpenRouter, basta colar sua API key:

/connect
# Selecione o provedor
# Escolha "Manually enter API Key"
# Cole sua chave

As chaves ficam armazenadas em ~/.local/share/opencode/auth.json

2. Inicialize o Projeto

Navegue até seu projeto e execute o comando /init para o OpenCode analisar o código e criar regras personalizadas.

cd /path/to/seu/projeto
opencode
/init

✅ O OpenCode analisa seu repositório
✅ Cria/atualiza o arquivo AGENTS.md com comandos de build, lint, teste e convenções
✅ Recomendado: versionar o AGENTS.md no Git

3. Selecione um Modelo

Execute /models para ver todos os modelos disponíveis nos provedores configurados.

/models

Selecione o modelo desejado com as setas e Enter. Você pode alternar entre modelos a qualquer momento.

✅ Dica: Use Ctrl+T para alternar entre variantes de pensamento (high/max reasoning) do modelo atual.

📝 Comandos Essenciais do TUI

Comando Ação Atalho
/connect Adicionar provedor de IA —
/init Criar/atualizar AGENTS.md —
/models Listar modelos disponíveis Ctrl+X M
/new Nova sessão Ctrl+X N
/undo Desfazer última mensagem Ctrl+X U
/redo Refazer alteração desfeita Ctrl+X R
/help Abrir ajuda —
/exit Sair do OpenCode Ctrl+X Q
