🐙 Integração com GitHub Actions

Automatize code review, triagem de issues e implementação de features diretamente do GitHub.

📋 Como Funciona

Mencione /opencode ou /oc em comentários de issues ou PRs, e o OpenCode executa tarefas no GitHub Actions runner.

🐛 Explicar Issue

"/opencode explique esta issue" → OpenCode lê o thread e explica

🔧 Corrigir Issue

"/opencode corrija isto" → Cria branch, implementa e abre PR

👀 Revisar PR

"/oc faça alterações aqui" → Comita mudanças no PR

📝 Revisar Linhas

Comente em linhas específicas na aba Files

🚀 Instalação Rápida

opencode github install

O comando guia você pela instalação do GitHub App, criação do workflow e configuração de secrets.

⚙️ Instalação Manual

1. Instale o GitHub App

Acesse github.com/apps/opencode-agent e instale no repositório alvo.

2. Workflow YAML

Adicione em .github/workflows/opencode.yml:

name: opencode
on:
  issue_comment:
    types: [created]
  pull_request_review_comment:
    types: [created]

jobs:
  opencode:
    if: |
      contains(github.event.comment.body, '/oc') ||
      contains(github.event.comment.body, '/opencode')
    runs-on: ubuntu-latest
    permissions:
      id-token: write
    steps:
      - uses: actions/checkout@v4
        with:
          fetch-depth: 1
          persist-credentials: false

      - uses: anomalyco/opencode/github@latest
        env:
          ANTHROPIC_API_KEY: ${{ secrets.ANTHROPIC_API_KEY }}
        with:
          model: anthropic/claude-sonnet-4-20250514

3. Secrets

Adicione a API key do seu provedor em Settings → Secrets and variables → Actions.

📝 Exemplos de Workflow

Code Review Automático

name: opencode-review
on:
  pull_request:
    types: [opened, synchronize]

jobs:
  review:
    runs-on: ubuntu-latest
    permissions:
      id-token: write
    steps:
      - uses: actions/checkout@v4
      - uses: anomalyco/opencode/github@latest
        env:
          ANTHROPIC_API_KEY: ${{ secrets.ANTHROPIC_API_KEY }}
        with:
          model: anthropic/claude-sonnet-4-20250514
          prompt: |
            Review este PR:
            - Verifique qualidade do código
            - Busque potenciais bugs
            - Sugira melhorias

Triagem de Issues (com filtro anti-spam)

name: Issue Triage
on:
  issues:
    types: [opened]

jobs:
  triage:
    runs-on: ubuntu-latest
    permissions:
      id-token: write
      issues: write
    steps:
      - name: Check account age
        id: check
        uses: actions/github-script@v7
        with:
          script: |
            const user = await github.rest.users.getByUsername({
              username: context.payload.issue.user.login
            });
            const days = (Date.now() - new Date(user.data.created_at)) / 86400000;
            return days >= 30;
          result-encoding: string

      - uses: actions/checkout@v4
        if: steps.check.outputs.result == 'true'
      - uses: anomalyco/opencode/github@latest
        if: steps.check.outputs.result == 'true'
        env:
          ANTHROPIC_API_KEY: ${{ secrets.ANTHROPIC_API_KEY }}
        with:
          model: anthropic/claude-sonnet-4-20250514
          prompt: |
            Revise esta issue. Se houver solução clara:
            - Forneça links de documentação
            - Adicione orientação de tratamento de erros
            Caso contrário, não comente.
