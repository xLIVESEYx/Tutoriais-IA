🔌 Provedores de IA

O OpenCode suporta 75+ provedores de LLM através do AI SDK e Models.dev. Use qualquer modelo de qualquer provedor.

📋 Visão Geral

Para adicionar um provedor:

  1. Execute /connect no TUI
  2. Selecione o provedor desejado
  3. Autentique via OAuth ou cole sua API Key
  4. Execute /models para ver os modelos disponíveis

As credenciais ficam armazenadas em ~/.local/share/opencode/auth.json

🆓 OpenCode Zen

O OpenCode Zen é um serviço de modelos curados e testados pela equipe do OpenCode. Você paga por uso e tem acesso a modelos validados para coding agents.

/connect
# Selecione "OpenCode Zen"
# Acesse https://opencode.ai/auth
# Crie uma API Key e cole no terminal

Modelos incluem: GPT 5.6 Sol/Terra/Luna, Claude Opus 4.8, Sonnet 4.5, Qwen 3.7 Max, Gemini 3.5 Flash e muitos outros.

⚙️ Configuração Personalizada

Para provedores não listados, use o formato de provider custom no opencode.json:

{
  "$schema": "https://opencode.ai/config.json",
  "provider": {
    "meu-provider": {
      "npm": "@ai-sdk/openai-compatible",
      "name": "Meu Provider Local",
      "options": {
        "baseURL": "http://localhost:8080/v1"
      },
      "models": {
        "meu-modelo": {
          "name": "Meu Modelo"
        }
      }
    }
  }
}
💡 Dica: Use opencode models para listar todos os modelos disponíveis e seus IDs exatos.
