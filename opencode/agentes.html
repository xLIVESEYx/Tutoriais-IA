🤖 Agentes

Configure e use agentes especializados para diferentes tarefas. Cada agente pode ter seu próprio modelo, prompt e permissões.

📋 Tipos de Agentes

🟢 Agentes Primários

Interaja diretamente com eles. Alterne com a tecla Tab.

Agente Função Permissões
Build Agente padrão de desenvolvimento. Acesso total a todas as ferramentas. allow
Plan Análise e planejamento. Read-only — não modifica arquivos. edit: ask, bash: ask

🔵 Subagentes

Invocados automaticamente ou manualmente com @nome.

Agente Função Acesso
General Pesquisas complexas e tarefas multi-passo. Pode modificar arquivos. Completo (exceto todo)
Explore Exploração rápida de código. Read-only, não modifica nada. Somente leitura
Scout Pesquisa em documentação externa e dependências. Somente leitura

⚙️ Agentes de Sistema (ocultos)

Agente Função
CompactionCompacta contexto automaticamente quando a janela está cheia
TitleGera títulos curtos para as sessões
SummaryCria resumos de sessão

🎮 Uso

Alternar entre agentes

Use Tab para ciclar entre Build e Plan.

Tab  → alterna entre agentes primários

Invocar subagentes

@general me ajude a pesquisar esta função
@explore encontre todos os arquivos que usam esta API

⚙️ Configuração de Agentes

Formato JSON

{
  "agent": {
    "code-reviewer": {
      "description": "Revisa código",
      "mode": "subagent",
      "model": "anthropic/claude-sonnet-4-20250514",
      "temperature": 0.1,
      "prompt": "Você é um revisor de código. Foque em segurança e performance.",
      "permission": {
        "edit": "deny",
        "bash": "deny"
      }
    }
  }
}

Formato Markdown

Coloque em ~/.config/opencode/agents/review.md:

---
description: Revisa código
mode: subagent
model: anthropic/claude-sonnet-4-20250514
temperature: 0.1
permission:
  edit: deny
  bash: deny
---

Você é um revisor de código experiente.
Foque em qualidade, bugs e segurança.

Opções Disponíveis

Opção Descrição Padrão

modelModelo específico para este agentemodelo global
temperatureControle de criatividade (0.0-1.0)0 (model-specific)
stepsMáximo de iterações agênticasilimitado
promptArquivo de system prompt customizado—
permissionPermissões específicas do agenteherdado do global
colorCor hexadecimal ou tema—
hiddenOcultar do menu @false
