⚙️ Configuração de Modelos

Modelo Padrão

{
  "$schema": "https://opencode.ai/config.json",
  "model": "opencode/gpt-5.1-codex",
  "small_model": "opencode/gpt-5-nano"
}

O small_model é usado para tarefas leves como geração de títulos.

Variantes de Pensamento

Configure variantes para alternar entre diferentes níveis de raciocínio com Ctrl+T:

{
  "provider": {
    "opencode": {
      "models": {
        "gpt-5": {
          "variants": {
            "high": {
              "reasoningEffort": "high",
              "textVerbosity": "low"
            },
            "low": {
              "reasoningEffort": "low",
              "textVerbosity": "low"
            }
          }
        }
      }
    }
  }
}

Timeout e Cache

{
  "provider": {
    "anthropic": {
      "options": {
        "timeout": 600000,
        "chunkTimeout": 30000,
        "setCacheKey": true
      }
    }
  }
}

🔄 Modelos Locais

Configure modelos locais para operação offline:

Ollama

{
  "provider": {
    "ollama": {
      "npm": "@ai-sdk/openai-compatible",
      "name": "Ollama (local)",
      "options": { "baseURL": "http://localhost:11434/v1" },
      "models": {
        "llama2": { "name": "Llama 2" }
      }
    }
  }
}
💡 Dica: Aumente num_ctx no Ollama para 16k-32k se tool calls não estiverem funcionando bem.
