💻 TUI / CLI / IDE / Web

O OpenCode está disponível em quatro interfaces. Escolha a que melhor se adapta ao seu fluxo.

⌨️ TUI (Terminal User Interface)

A interface principal do OpenCode. Basta executar opencode no terminal.

Comandos Slash

Comando Ação Atalho
/connectAdicionar provedor—
/initCriar AGENTS.md—
/compactCompactar sessãoCtrl+X C
/modelsListar modelosCtrl+X M
/newNova sessãoCtrl+X N
/sessionsListar sessõesCtrl+X L
/undoDesfazer alteraçõesCtrl+X U
/redoRefazer alteraçõesCtrl+X R
/shareCompartilhar sessão—
/exportExportar como MarkdownCtrl+X X
/editorAbrir editor externoCtrl+X E
/helpAjuda—
/exitSairCtrl+X Q

Funcionalidades TUI

  • @arquivo — Referencie arquivos com busca fuzzy
  • !comando — Execute comandos shell diretamente
  • Ctrl+T — Alterne variantes de modelo (thinking)
  • Tab — Alterne entre agentes Build/Plan
  • Arraste imagens para adicionar ao prompt
  • Paleta de comandos com Ctrl+P

Flags de Inicialização

opencode                         # Inicia TUI no diretório atual
opencode /path/to/projeto        # Inicia em diretório específico
opencode -c                      # Continua última sessão
opencode -s          # Continua sessão específica
opencode --auto                  # Auto-approve permissions
opencode --model provider/model  # Modelo específico
opencode --agent plan            # Agente específico

📟 CLI (Command Line Interface)

Comandos para automação e scripting:

Comandos Principais

Comando Descrição
opencode run "prompt"Modo não-interativo (óptimo para scripts)
opencode serveServidor headless (API HTTP)
opencode webServidor com interface web
opencode attach <url>Conectar TUI a servidor remoto
opencode session listListar sessões
opencode statsEstatísticas de uso e custos
opencode modelsListar modelos disponíveis
opencode auth loginConfigurar credenciais
opencode agent createCriar novo agente interativamente
opencode mcp addAdicionar servidor MCP
opencode plugin <module>Instalar plugin npm
opencode upgradeAtualizar OpenCode

Modo Run com Attach

Evite cold boot de MCP servers conectando a um servidor existente:

# Terminal 1: servidor
opencode serve

# Terminal 2: executa comandos
opencode run --attach http://localhost:4096 "Explique closures"

🔌 IDE (VS Code / Cursor / Windsurf)

Instalação Automática

Abra o terminal integrado do VS Code e execute opencode. A extensão instala automaticamente.

# No terminal integrado do VS Code:
opencode

Atalhos IDE

Atalho Ação
Cmd+Esc (Mac) / Ctrl+EscAbrir OpenCode em terminal dividido
Cmd+Shift+Esc (Mac) / Ctrl+Shift+EscNova sessão OpenCode
Cmd+Option+K (Mac) / Alt+Ctrl+KInserir referência de arquivo (@File#L37-42)
💡 Editor externo: Configure EDITOR para usar seu editor favorito com /editor e /export: export EDITOR="code --wait"

🌐 Web UI

Iniciar Interface Web

opencode web

Abre automaticamente no navegador. Suporta sessões, servidores e mais.

Configurações Web

Flag Descrição
--port 4096Porta específica
--hostname 0.0.0.0Acessível na rede
--mdnsDescoberta mDNS (opencode.local)
--cors https://exemplo.comOrigens CORS adicionais
OPENCODE_SERVER_PASSWORD=secretAutenticação básica

Attach TUI + Web

Use TUI e Web simultaneamente compartilhando o mesmo servidor:

# Terminal 1: servidor web
opencode web --port 4096

# Terminal 2: attach TUI
opencode attach http://localhost:4096
