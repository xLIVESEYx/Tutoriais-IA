O que é o OmniRoute?

O OmniRoute é um gateway de IA open-source que conecta 265 providers de IA através de um único endpoint local (http://localhost:20128/v1). Ele oferece fallback automático entre providers, compressão de tokens (economiza 15-95%), suporte a 90+ providers gratuitos, e integração perfeita com Claude Code, Codex, Cursor, Cline, Copilot, OpenCode e outros agentes de IA.

💡 Por que usar o OmniRoute?
Pare de desperdiçar dinheiro com quotas não utilizadas, limites de taxa que interrompem seu trabalho, e configuração repetitiva de cada ferramenta. O OmniRoute gerencia tudo automaticamente.

📸 Dashboard Principal

Interface web completa para gerenciar providers, combos, analytics, logs e muito mais.

🚀 Quickstart de 3 Passos

1. Instale o OmniRoute

npm install -g omniroute
omniroute

Dashboard abre em http://localhost:20128 e API em http://localhost:20128/v1

2. Conecte Providers e Crie API Key

  1. Abra Dashboard → Providers e conecte pelo menos um provider (OAuth ou API key)
  2. Abra Dashboard → Endpoints e crie uma API key
  3. (Opcional) Configure seu combo de fallback em Dashboard → Combos

3. Configure Sua Ferramenta de IA

Base URL: http://localhost:20128/v1
API Key:  [copie do Dashboard → Endpoints]
Model:    if/kimi-k2-thinking (ou qualquer prefix provider/model)
