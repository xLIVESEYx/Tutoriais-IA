📋 Passo a Passo: 4 Passos para $0

Configure estes 4 providers e use o combo "Free Stack" template:

Passo 1: Conectar Kiro (AWS Builder ID OAuth)

Dashboard → Providers → Kiro → Connect
→ Login com conta AWS → Authorizar
✅ Claude Sonnet 4.5/Haiku 4.5 ilimitado

Passo 2: Conectar Qoder (Google OAuth)

Dashboard → Providers → Qoder → Connect
→ Login com Google → Authorizar
✅ Kimi K2, Qwen, DeepSeek ilimitado

Passo 3: Conectar Qwen (Device Code)

Dashboard → Providers → Qwen → Connect
→ Copie o código → acesse https://qwen.ai/code
→ Cole o código → Authorizar
✅ Qwen3-Coder-Plus ilimitado

Passo 4: Usar o combo Free Stack

Dashboard → Combos → Templates → "Free Stack ($0)"
→ Ative o combo
→ Use model: if/kimi-k2-thinking ou auto

✅ Tudo funcionando com $0 de custo!

⚠️ Aviso sobre Termos de Serviço

Os providers gratuitos podem alterar seus termos!
Sempre verifique as políticas no site do provider. Alguns podem exigir verificação de identidade. O OmniRoute não controla os tiers gratuitos — apenas os expõe através do endpoint unificado.

Os seguintes providers têm flags de termos que você deve ter em mente:

  • Vertex AI (Google) — pode exigir verificação de identidade
  • Together AI — free tier pode ter rate limits agressivos
  • DeepSeek — $50 créditos iniciais mas pode exigir pagamento após
